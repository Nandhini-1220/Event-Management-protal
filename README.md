# Event Management Portal (Frontend)

A modern, responsive **Event Management Portal frontend** built using only HTML, CSS, and JavaScript.  
This project simulates a real-world event discovery and management platform with a clean UI and smooth interactions.

---

## 📌 Features

- 🎯 Modern landing page with hero section
- 📅 Event listing with card-based UI
- 🔍 Simple event filtering (JavaScript)
- 📖 Event details modal / section view
- 👤 Basic dashboard UI (mock data)
- 📱 Fully responsive design (mobile + desktop)
- ✨ Smooth hover effects and animations
- 🎨 Clean SaaS-style UI design

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript (ES6)

## 🎨 UI Highlights

- Modern SaaS-style design
- Clean spacing and typography
- Interactive event cards
- Responsive grid layout
- Simple animations and transitions
- Dark/soft theme styling
